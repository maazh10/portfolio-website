# Personal Portfolio

My personal website & blog built with [Astro](https://astro.build)

**Live:** https://maazh.tech  

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Docker

Static site is built using Docker and served with nginx on a Google Cloud VM. For local proxy testing:

```bash
docker-compose up
```
