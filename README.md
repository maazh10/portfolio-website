# Personal Portfolio

Personal site built with [Astro](https://astro.build)

**Live:** https://maazh.tech  

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Docker

Static site is built in the `Dockerfile` and served with nginx. For local proxy testing:

```bash
docker-compose up
```
